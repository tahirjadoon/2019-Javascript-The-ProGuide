Tasks

  • Query the section with the ID of main-section. Then pick the UL using the next sibling and then pick its first element child to pick the li
  • UL list is inside the section with the ID of main-section, apply the red-bg and visible classes to the section as default.
  • Select the section and apply the visible class by default to it.
  • Add an event listener to the button and when it is clicked apply the visible and invisible class. Also look into toggling the invisible class only.

const button = document.querySelector('#toggleUlLi');
const buttonBg = document.querySelector('#toggleBg');
button.addEventListener('click', () => {
 //either using the .className check which classes are applied and then reapply the classes
 /*

  section.className = 'red-bg invisible';
 } else {
  section.className = 'red-bg visible';
 }
 */
 //or toggle the invisible class
 section.classList.toggle('invisible');

 //toggle the bg button as well
 buttonBg.classList.toggle('invisible');
});

buttonBg.addEventListener('click', () => {
 //toggle the bg color for the section
 section.classList.toggle('blue-bg');
});
