Task

  • Create three radio buttons for the player choice for ROCK, PAPER and SCISSORS
  • Create a Game Start Button and create an event listener to it in the JS file.
  • Create an object with ROCK, PAPER, SCISSORS choices and also setup a default choice.
    Also, create winner choices in the same object
  • Write a player choice function, use any method that we have learned so far. It will ask the user for a choice and then in case of invalid selection will select the default choice by alerting the user otherwise will return the choice.
  • Create an event listener with anonymous function for the game start button, call the above choice function and then console.log the choice.
  • Open dev tools to view console.log results
  • Write another function which will generate a randon value.
    RandomValue < 0.34 should return ROCK
    RandomValue < 0.67 should return PAPER
    For the default return SCISSORS
  • Write a function to determine a winner.
